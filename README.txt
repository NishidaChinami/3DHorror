【作品について】
・本作品のコードで使用しているdxlib_extファイルのコードは学校の講師によって配布されたものであり、そちらを一部使用して作品制作を行いました。
・チュートリアルからプレイしていただくことをお勧めします。
・マップ読み込みに少々時間がかかります。

【力を入れたところ】
①音や光など五感に伝わる表現
・ホラーゲームにおいて音と光は重要な演出であるため特にこだわりました。
音に関しては、廃ビル内では音が反響するようになっており、また敵のいる向きから音がなる、また距離によって音が大きくなるようにしました。光に関しては、敵がいるところは赤く光るようになっており、近いと赤い光は強くなります。また、微かに点滅しており、光が揺れているような表現にしました。これらの表現でプレイヤーになにかが近づいてくるという恐怖を演出しました。
②敵の強さ
・敵についても視野と聴覚を持たせており、視野に移るか一定範囲内でプレイヤーが走るとプレイヤーとの最短経路をとって追いかけてくるようにしました。また、プレイヤーを見つけていないときは、脱出に必要なキーアイテムの場所を巡回しており、プレイヤーとの接触する確率が上がるようにしました。
この一連も動作の実装は、一番時間がかかっておりプログラム面でも努力し点です。
③ステージの自動生成
・ステージについても、力を入れました。ステージは穴掘り法という自動生成を実装しており、ランダムな迷路になっています。このため、一度ゲームオーバーになってゲームをやり直しても、難易度は変わらず何度も迷子になるようなステージづくりを意識しました。
④チュートリアルステージの導入
・メインのステージで、画面に文字や説明を入れたくないが、ユーザービリティ面を下げないためにチュートリアルステージを作成しました。また、チュートリアルステージでは、これからホラーゲームが始まるという雰囲気の演出にも意識をいれました。
